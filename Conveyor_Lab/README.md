# Conveyor Lab

롤러 컨베이어 **실험·실행 환경**을 한곳에 모은 폴더입니다. Blender 시뮬 코드(`src/blender_sim/conveyor/`)는 레포 패키지로 그대로 두고, 여기서는 호스트 스크립트·문서·기본 산출 경로만 다룹니다.

| 항목 | 경로 |
|------|------|
| 실행·익스포트 스크립트 | `Conveyor_Lab/scripts/` |
| 안내 문서 | [docs/CONVEYOR.md](docs/CONVEYOR.md) |
| 기본 MP4 등 출력 | `Conveyor_Lab/outputs/` (로컬 산출물은 git에서 제외) |

루트의 `scripts/run_conveyor_demo.py` / `scripts/export_conveyor_glb.py` 는 이 폴더의 스크립트로 연결되는 호환용 진입점입니다.
