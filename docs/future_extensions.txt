향후 확장 후보 (필수 아님 — 기록용)

현재 기본선: Blender에서 조명·카메라·물리·렌더 도메인 다양화 + Python 2D 증강 + YOLO 검출·추적·(선택) 분류.



필요할 때만 검토할 것들:
- 실제 이미지로도 검증
- 컨베이어벨트 바꾸기
- 꼭지 넣어서 만들기
- 롤러/베이어에 디테일한 물리 법칙 — 마찰·굴림 저항·접촉/구동 모델·토크·서브스텝 정밀도 등




- Diffusion / 인페인팅 (예: DGA2 유사) — 배경·조명만 변경 시 병변·과일 마스크로 보호 권장

- 2단계 세그 (Mask R-CNN + U-Net++ 등) — 픽셀/인스턴스 라벨 또는 합성 마스크 export 선행

- 배포 가속 — ONNX / TensorRT / FP16
