# 감귤 병해 시각 합성 — 참고 요약 (합성용)

출처: UC IPM, UF/IFAS CH159, 일반 병리 설명. 합성 재질은 **교육/합성 데이터용 근사**이며 진단 기준이 아님.

| 클래스 (폴더명 예) | 시각적 단서 (과피) | 합성 시 반영 |
|-------------------|-------------------|--------------|
| **healthy** | 균일한 주황·황금 껍질, 명확한 병변 없음 | 따뜻한 오렌지 베이스, 미세 노이즈만 |
| **Canker** (궤양병) | 물침범 느낌 가장자리, **황색 할로**, 중심은 갈색·눌린 듯 | 할로(노랑)·중심(갈색) 대비 큰 반점 |
| **Scab** (총채병) | **콕 마른 융기**, 코르크질, 작은 원형 융기가 뭉치기도 | 고주파 반점·밝은 융기색 |
| **Black spot** | **작은 함몰 반점**, 가장자리 벽돌색·검정, 할로 가능 | 어두운 중심 + 붉은 링 형태 반점 |
| **Greening** (황룡병) | **비대칭** 황록·황색 **얼룩** (잎·과피 불균일) | 녹·황 혼합 얼룩 마스크 |

UV는 **한 덩어리에 가깝게** 쓰기 위해 구면 투영을 우선 시도해 사진 한 장이 찢어져 보이는 현상을 줄입니다.
